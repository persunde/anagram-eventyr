# Anagram

A Kotlin project.

## Prerequisites

- JDK 11 or higher
- Maven 3.6+

## Build

```bash
mvn package
```

This compiles the Kotlin source and produces `target/anagram-1.0-SNAPSHOT.jar`.

## Run

```bash
java -jar target/anagram-1.0-SNAPSHOT.jar
```

Expected output:

```
at ta
bar bra
bry byr
dem med
den ned
denne enden
...
```

## Run tests

```bash
mvn test
```
